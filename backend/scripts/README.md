# Scripts Directory

This directory contains utility scripts organized by purpose:

## `/dev/` - Development Scripts
- `security_demo.py` - Security feature demonstrations
- `security_final_demo.py` - Final security implementation demo
- `security_test_runner.py` - Security test automation

## `/admin/` - Administrative Scripts
- `delete_all_users.py` - User cleanup utility
- `clear_cached_users.py` - Cache management utility

## `/deployment/` - Deployment Scripts
- (Future deployment automation scripts)

## Usage
Run scripts from the backend root directory:
```bash
python scripts/admin/clear_cached_users.py
python scripts/dev/security_demo.py
```
